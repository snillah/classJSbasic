Array

Array is storing the value in continues order

    Fixed Array

  • Some language as still fixed array like => c,c++,golang eg: int [5] ={1,2,3,4,5}
  • Single DataType can store single array:
    • int 4byte
    • float 4byte
    • double 8byte
    • char 1byte

    Dynamic Array

  • Some language as still dynamic array like => java-stript, python eg: below
  • Different DataType array store in single array itself

    • eg: var arr = [1, "saran", true, {language: ["Tamil","English"]},playing=["chess","footbal","cricket"]]

    Result:

    [ 1, "saran", true, { "language": [ "Tamil", "English" ] }, [ "chess", "footbal", "cricket" ] ]

    0-number: 1
    1-string: "saran"
    2-boolean: true
    3-object: {language: Array(2)}
    4-array: (3) ['chess', 'footbal', 'cricket']
    length: 5
